# Store assets (Google Play, etc.)

## Feature graphic (Google Play)

- **Size:** 1024 × 500 px
- **File:** `feature_graphic.png` in this folder
- **Use:** Upload as “Feature graphic” in Play Console (App store listing → Main store listing).

Use a horizontal banner with game art and your app name (“Moving Brick”). Keep important content in the center; edges can be cropped on some devices.

## Optional

- **App icon:** 512 × 512 px for store listing (can use the same as `assets/icon/app_icon.png`).
- **Screenshots:** 16:9 or 9:16, min length 320 px (e.g. 1080×1920 for phone).
